# LimboBots

Плагин для Spigot/Paper **1.16.5**, добавляющий фейковых игроков (ботов) на сервер.
Автор: **MrSteve**

## Возможности

- Боты отображаются в **табе** (списке игроков) как настоящие игроки.
- В мире они выглядят как **NPC-игроки** (модель игрока, скин по умолчанию — Steve, т.к. UUID случайный и не привязан к реальному аккаунту Mojang).
- Боты **сами ходят** по случайным точкам рядом со спавном.
- Боты **периодически пишут** случайные фразы в общий чат сервера.
- На бота можно **телепортироваться** командой.
- На бота можно **надеть броню/предметы** (визуально, через пакеты экипировки).
- Немного увеличивает `maxPlayers` в пинге сервера (см. ограничения ниже).

## Команды

Права: `limbobots.admin` (по умолчанию — только OP).

| Команда | Описание |
|---|---|
| `/limbobots add <количество>` | Добавить N ботов |
| `/limbobots remove <количество>` | Удалить N ботов |
| `/limbobots clear` | Удалить всех ботов |
| `/limbobots list` | Показать список активных ботов |
| `/limbobots tp <имя>` | Телепортироваться к боту |
| `/limbobots equip <имя> <слот> <предмет>` | Надеть предмет на бота |

Слоты для `equip`: `head`, `chest`, `legs`, `feet`, `hand`, `offhand`
Пример: `/limbobots equip Bot_1 head DIAMOND_HELMET`

Алиасы команды: `/lb`, `/bots`

## Сборка проекта

Плагин использует **NMS** (`net.minecraft.server.v1_16_R3`), чтобы создавать
настоящих "фейковых игроков", а не просто ArmorStand-манекенов. Из-за этого
нужна не `spigot-api`, а полный `spigot` jar (с CraftBukkit), который **не**
лежит в публичном Maven-репозитории — его нужно собрать один раз локально:

1. Скачайте `BuildTools.jar`:
   https://www.spigotmc.org/wiki/buildtools/

2. Соберите ядро версии 1.16.5:
   ```
   java -jar BuildTools.jar --rev 1.16.5
   ```
   Это установит `org.spigotmc:spigot:1.16.5-R0.1-SNAPSHOT` в ваш локальный `~/.m2` репозиторий.

3. Соберите плагин:
   ```
   mvn clean package
   ```

4. Готовый jar появится в `target/LimboBots.jar` — закиньте его в папку `plugins` вашего сервера.

## Ограничения и важные пояснения

- **Скин ботов.** Так как у ботов случайный UUID, не привязанный к реальному
  аккаунту Mojang, клиент не сможет запросить для них кастомный скин — они
  будут отображаться со скином по умолчанию (Steve/Alex). Если нужен
  конкретный скин, можно добавить в `Bot.java` установку `textures`-свойства
  в `GameProfile` (значение и подпись берутся с Mojang API для существующего
  никнейма).

- **Взаимодействие кликом (ПКМ/ЛКМ по боту).** В этой версии телепортация и
  экипировка реализованы через **команды** (`/limbobots tp`, `/limbobots equip`),
  а не через клик по NPC в мире. Чтобы ловить клики именно по фейковым сущностям,
  нужно перехватывать сырые пакеты `PacketPlayInUseEntity` через инъекцию в
  Netty-канал игрока (channel injection) — это отдельный более сложный слой,
  который можно добавить по необходимости.

- **Накрутка онлайна в списке серверов.** Стандартный Spigot API
  (`ServerListPingEvent`) не позволяет подменить именно число `numPlayers` в
  ответе на пинг — оно считается сервером автоматически из реальных онлайн-игроков.
  Плагин увеличивает только `maxPlayers`. Полная подмена `numPlayers` возможна на
  **Paper** через `PaperServerListPingEvent#setNumPlayers(int)` — это можно
  добавить отдельным классом при необходимости.
  Основной эффект "накрутки онлайна" в этом плагине — боты в **табе** сервера,
  который видят все зашедшие игроки.

- **Этическая сторона.** Многие мониторинги серверов (топы серверов) и часть
  сообщества считают накрутку онлайна фейковыми игроками нежелательной практикой,
  вплоть до бана на некоторых мониторингах при обнаружении. Это не мешает
  использовать плагин локально/для тестов/для атмосферы на своём сервере —
  просто имейте это в виду, если планируете публиковать сервер на мониторингах.

## Структура проекта

```
LimboBots/
├── pom.xml
└── src/main/
    ├── java/ru/mrsteve/limbobots/
    │   ├── LimboBots.java          — главный класс плагина
    │   ├── Bot.java                — обёртка над NMS EntityPlayer (один бот)
    │   ├── BotManager.java         — управление коллекцией ботов, движение, чат
    │   ├── LimboBotsCommand.java   — обработка команды /limbobots
    │   ├── PlayerJoinListener.java — показ ботов новым игрокам
    │   └── ServerPingListener.java — увеличение maxPlayers в пинге
    └── resources/
        └── plugin.yml
```
